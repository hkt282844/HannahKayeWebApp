# HannahKayeWebApp
Simple web app for Hannah Kaye music references

### Prerequisites
* Python3
* Python Environment
* Flask & pacakges: __pip install flask flask-sqlalchemy flask-login__
* SQLite database: run __python init_db.py__
* __config.py__ file containing **SECRET_KEY**
